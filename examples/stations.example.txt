# FT-847 quick-tune presets
# Format: name, frequency_hz, mode, shift, offset_hz, tone_hz
#   name       - no spaces, used on the command line
#   frequency  - Hz
#   mode       - LSB USB CW CWR AM FM AMN FMN CWN CWNR
#   shift      - '+' '-' or 'S' (simplex)
#   offset_hz  - repeater shift amount, e.g. 600000 for 600 kHz (0 if simplex)
#   tone_hz    - CTCSS tone e.g. 88.5, or NONE
#
# Examples below are placeholders -- replace with your own stations.
# GB3TS is a UK example (2m FM repeater, 600kHz shift, tone 118.8Hz)

GB3TS, 145737500, FM, +, 600000, 118.8
GB3NT, 145762500, FM, +, 600000, NONE

# Weatherfax / marine weather broadcast example (SSB, simplex, no tone)
WMC_WEATHERFAX, 12786000, USB, S, 0, NONE

# Aviation / ATC monitoring example (AM, simplex)
# Note: FT-847 airband coverage is receive-only per its rx range (108-174 MHz)
ATC_APPROACH, 124200000, AM, S, 0, NONE

# ISS / satellite downlink example (FM, simplex)
ISS_APRS, 145825000, FM, S, 0, NONE

# Crossband net example: format is
#   name, CROSSBAND, rx_freq_hz, rx_mode, tx_freq_hz, tx_mode, tx_tone_hz, note
# Uses the FT-847's Satellite mode (independent SAT RX / SAT TX VFOs) since
# CAT can't drive true split on the Main VFO. RX 2m, TX 70cm.
NIGHT_NET, CROSSBAND, 147475000, FM, 439225000, FM, NONE, Nightly crossband net

# SO-50 needs a TWO-STEP tone sequence, not just one tone -- these are two
# separate presets sharing the same frequencies. Key up briefly (~2 sec)
# on SO50_ARM first to start its 10-minute on-timer, THEN switch to
# SO50_QSO for your actual transmissions. Sending the arm tone again
# within the window resets the timer. Downlink drifts +/-9-10kHz over a
# pass (Doppler) -- this preset gets you on frequency at the start, but
# doesn't track continuously through the pass.
SO50_ARM, CROSSBAND, 436795000, FM, 145850000, FM, 74.4, SO-50: key 2sec to arm 10min timer
SO50_QSO, CROSSBAND, 436795000, FM, 145850000, FM, 67.0, SO-50: use for actual QSO after arming

# ISS crossband voice repeater (single tone, no arming needed -- but
# check current ISS repeater status before a pass, it's been turned off
# for extended periods in the past due to onboard equipment conflicts)
ISS_XBAND, CROSSBAND, 437800000, FM, 145990000, FM, 67.0, ISS crossband repeater

# HF weatherfax example: "FAX" is a mode alias -- resolves to USB plus the
# standard -2kHz listen-below-carrier offset automatically, so no extra
# fields are needed for the common case. frequency_hz stays the published
# schedule frequency.
WEFAX_7535, 7535000, FAX, S, 0, NONE

# If a particular fax station needs a different offset than -2kHz, add it
# explicitly (8-field form) -- this overrides the FAX default of -2000.
WEFAX_CUSTOM, 5000000, FAX, S, 0, NONE, -1900, Uses -1.9kHz instead of the -2kHz default
